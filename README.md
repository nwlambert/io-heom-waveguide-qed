# io-heom-waveguide-qed
Examples on how to use the io-heom method for waveguide qed problems.

These are Jupyter notebooks using a local version of the QuTiP HEOM solver (found in /heom_funcs) modified to accomodated input-output baths.  This is a static repository to go alongside the acompanying article, and will be superseded by the incorporation of the input-output baths in QuTiP Core. As such, this repository depends on a very specific version of QuTiP version 5.2.1.  It will not work with newer (or potentially older) versions.

The three examples are:

- nonmarkovianwaveguide-ohmic.ipynb
- nonmarkovianwaveguide-lorenz.ipynb
- nonmarkovianwaveguide-cavityarray.ipynb
